# Starter Playbook Library

This folder contains the first draft playbook objects for Pattern Bridge.

These are not final.
They are starter objects designed to pressure-test:
- the playbook schema
- the ontology
- variant handling
- translation logic
- failure-mode capture

## Initial playbook set

1. Auction Imbalance Continuation
2. LVN Island Reversal
3. Wedge Break Reversal
4. Cave Fill Reclaim
5. Elastic Band Reversal
6. Caught in the Hole
7. Block Reversal
8. Zeros Reversal

## Notes

- These are playbook-family level objects, not single trade examples.
- Some are more profile-led, some more order-flow-led, and some are hybrids.
- Several likely need splitting into cleaner variants later.
- The point right now is not perfect taxonomy. The point is getting real objects into the repo.
