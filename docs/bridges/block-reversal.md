# Block Reversal

## Summary
A reversal bridge where aggressive directional action runs into an area that suddenly turns heavy, blocks progress, and then flips as the move fails to continue.

## Status
Draft

## Family
Order-flow-led reversal

## Brand
The move slams into a wall, stalls, turns heavy, and the block becomes the pivot.

## Context

### Regime
- directional push into destination
- mature intraday move
- live execution-driven reversal opportunity

### Event types
- none
- post-news reactions
- session extensions into reference

### Session structure
- often appears intraday at or after a push into a meaningful area

### Higher timeframe conditions
- much better when block occurs at a meaningful reference
- weaker when taken in the middle of nowhere

### Structural references
- destination level
- daily support/resistance
- cave edge
- profile boundary
- prior session key zone

### Participation conditions
- enough pace to create a real push into the block
- reversal quality improves when original move has already expended effort

## Pattern definition

### Preceding conditions
- directional attack into a reference
- aggressive flow fails to continue smoothly
- price action turns heavy at the edge

### Mechanics
1. move attacks a level
2. progress stalls and starts to feel blocked
3. failed continuation exposes trapped participants
4. reversal develops from the block zone

### Identifier
- heaviness or inability to push beyond a zone
- repeated failure to extend despite aggressive effort
- visible change in pace or absorption behavior

### Confirmation
- loss of follow-through in original direction
- reversal-side pace improves
- block zone holds as pivot

### Expected progression
- early reversal may still be messy
- best cases clean up quickly once the trap is exposed
- continuation often depends on context beyond the block itself

### Target logic
- move back through recent intraday structure
- reclaim of prior balance or rotation objective
- larger reversal target if broader context aligns

### Invalidation
- block fails and original direction breaks through cleanly
- reversal cannot hold the pivot zone

### Failure modes
- mistaking temporary pause for real block
- fading a strong trend too early
- taking the trade without meaningful destination context

### Execution notes
- often highly execution-sensitive
- can be improved by pairing with delta, profile, or larger structural read
